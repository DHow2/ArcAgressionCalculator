# Arc

This is a small tool for the game ARC Raiders to help you save your Round Scores and (Potentially) give you predicted next Lobby. It works by taking your last 10 games into consideration. 

I made this Project for myself to improve my coding skills for my apprenticeship (even though I am doing the German "Fachinformatiker, Systemintegration, which is the non programmer version here. But we still need to do some coding in school.), but I thought it would be nice to share it with the community. If you have any suggestions or want to contribute, feel free to open an issue or a pull request.
